# Aplikacja chmurowa WPF

Aplikacja WPF umożliwiająca zarządzanie plikamai użytkownika.

## Technologie

- C#
- WPF
- Entity Framework
- SQLite

## Funkcjonalności

- rejestracja i logowanie użytkowników
- autoryzacja użytkowników
- przesyłanie plików
- zarządznie plikami
